Readonly client for the Twitch API.

Currently Twitch API version 3 is used.

By using the Twitch Client you agree to follow the Twitch API Terms of Service and the Twitch Terms of Service. This library is in no way affiliated with, authorized, maintained, sponsored or endorsed by Twitch or any of its affiliates or subsidiaries

Examples

use twitch_client::*;

let twitch_client = TwitchClient::new();
let top_games = twitch_client.top_games(TopGamesParams::default()).unwrap();
println!("Total games: {}", top_games.total());
